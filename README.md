
# Blobs
Recreation of the card-game Blobs to be played multiplayer in a browser.

**Technologies Used**

 - NodeJS & Express
 - Sockets.io
 - AngularJS
 - Less
 - Grunt

**Getting Started**
After cloning/downloading the project install it using

    npm install
Then run the project using

    npm start
Your game server will be available on port 3000
